# 技术设计文档

## 技术方案：PWA Web App

选择 PWA 的理由：
- 零成本分发，无需 iOS 证书
- Safari 添加到主屏幕，体验接近原生
- 随时更新，无需审核
- 跨平台，手机/平板/电脑都能用

---

## 技术栈

| 层级 | 选型 | 理由 |
|------|------|------|
| 前端框架 | Vue 3 + Vite | 轻量快速，适合小项目 |
| UI 样式 | 纯 CSS / 手写 | 保持轻量，不引入 UI 库 |
| 语音朗读 | Web Speech API (SpeechSynthesis) | 浏览器原生，零成本，离线可用 |
| AI 故事生成 | 后端代理调用 LLM API | 前端不暴露 API Key |
| 后端服务 | Node.js (Express) | 轻量，仅做 API 代理 |
| 数据存储 | localStorage / IndexedDB | 本地存储，无需数据库 |
| 部署 | Vercel / Cloudflare Pages | 免费，自带 HTTPS，支持 PWA |

## 页面结构

```
/ (首页 - 故事创作)
  ├── 宝宝名字输入
  ├── 关键词选择/输入
  ├── 故事类型选择
  ├── 故事时长选择
  └── "生成故事" 按钮

/story (故事展示 + 朗读)
  ├── 故事标题
  ├── 故事正文（大字体）
  ├── 朗读控制（播放/暂停/语速）
  └── 收藏按钮

/favorites (收藏列表)
  ├── 收藏的故事列表
  └── 点击可重新阅读/朗读
```

## Prompt 设计（核心）

```
你是一位温柔的中文儿童故事作家，专门为 {age} 岁的小朋友写睡前故事。

请根据以下信息创作一个睡前故事：
- 主角名字：{name}（如果没有则用"小朋友"）
- 关键词/主题：{keywords}
- 故事类型：{type}
- 故事长度：约 {length} 字

要求：
1. 语言简洁温柔，适合幼儿理解，避免复杂词汇
2. 故事结构完整：开头引入 → 小冒险或小问题 → 解决 → 温馨结尾
3. 结尾自然融入一个小道理，不要说教
4. 可以融入中国文化元素
5. 整体氛围温馨安宁，适合睡前场景
6. 请给故事起一个可爱的标题

请直接输出故事，格式：
标题：xxx
正文：xxx
```

## LLM API 选型建议

| 服务 | 优势 | 成本 | 国内可用 |
|------|------|------|---------|
| DeepSeek | 便宜，中文好 | ~0.001元/故事 | ✅ |
| 通义千问 | 阿里云，稳定 | 有免费额度 | ✅ |
| OpenAI GPT-4o-mini | 质量高 | ~0.01元/故事 | ❌ 需代理 |
| Claude Haiku | 质量高 | ~0.01元/故事 | ❌ 需代理 |

**推荐 MVP 阶段用 DeepSeek 或通义千问**，国内直连，成本极低。

## 开发计划

### Phase 1 — MVP（核心可用）
1. 前端：首页（创作入口）+ 故事展示页 + 收藏页
2. 后端：故事生成 API（代理 LLM）
3. TTS：Web Speech API 朗读
4. 存储：localStorage 收藏功能
5. PWA：manifest + service worker

### Phase 2 — 体验优化（后续）
- 更好的 TTS 语音（接入第三方语音合成）
- 故事生成的流式输出（打字机效果）
- 更多关键词预设和主题模板
- 离线缓存已收藏的故事
